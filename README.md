# 🎵 Music Wave

Music Wave is a responsive music streaming web application built using HTML, CSS, and JavaScript. The project provides playlist exploration, artist pages, login/signup authentication flow, and a modern music website interface.

---

## 🚀 Features

- 🎧 Modern Music Website UI
- 🔐 Login & Signup Pages
- 📂 Playlist Exploration
- 🎵 Multiple Artist & Song Pages
- 📱 Responsive Design
- 🎨 Interactive Hover Effects
- 📊 Google Analytics Integration
- 🖼️ Album & Song Collections

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap Icons

The objective of Music Wave is to create a visually appealing music streaming platform interface where users can explore playlists, artists, and albums through a clean and interactive frontend experience.
